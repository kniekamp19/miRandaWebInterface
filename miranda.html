miRanda: microRNA Target Detection

Welcome to the miRanda miRNA Target Detection web interface! miRanda is an algorithm for finding genomic targets of microRNAs. Local download is available here and the manpage is available here.

To run miRanda, upload miRNA query and genomic reference files of sequences in FASTA format.
